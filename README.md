Segment3D intersection task

This repository contains code for the Segment3D intersection task, which is a part of the 3D computer vision domain. The task involves determining whether two 3D segments intersect and, if so, finding the point of intersection.
The code of the solutiion is in ConsoleApplication1.cpp file. You can compile and run the code to test its functionality. Or you can use the test provided in the file titled test.txt to verify the correctness of the implementation.
